## Adding Stories

1. Create a new file in `_stories/` with front matter:
   ```markdown
   ---
   title: Story Title
   date: 2026-01-08
   excerpt: Brief description
   ---
   
   Story content here...
   ```

2. Commit and push - it will appear automatically on the site
